# BlinkenClock
A Wall-Clock based on a RGB-LED Stripe.
Inspired by [Bjoern Knorr](http://netaddict.de/blinkenlights:blinkenclock).

* RGB-LED Stripe with 60 LEDs/Pixels
* LDR Light Sensor
* DS1307 RTC (Real-Time-Clock) with backup battery
